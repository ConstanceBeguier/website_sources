AWS Tower has four key features:
  • 2. Discover: This feature allows you to identify and discover the assets (such as EC2 instances, S3 buckets, and IAM roles) that exist within your AWS account.
  • 3. Audit: This feature allows you to perform an audit of your assets, which involves checking them for potential security and compliance issues.
  • 4. IAM: This feature is connected to the Identity and Access Management (IAM) service in AWS, which enables you to investigate AWS IAM roles in more detail
  • 5. Draw: This feature allows you to create visual diagrams or drawings of your assets, which can be helpful for understanding the relationships and dependencies between them.
