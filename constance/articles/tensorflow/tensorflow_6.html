TensorBoard

TensorBoard est un outil pour visualiser l'apprentissage ce qui peut faciliter l'optimisation d'un modèle.

Comment utiliser TensorBoard

Pour utiliser TensorBoard, il suffit d'utiliser le callback TensorBoard lors de l'apprentissage.

# TensorBoard
from datetime import datetime
log_dir = "logs/" + datetime.now().strftime("%Y%m%d-%H%M%S")
tensorboard_callback = tf.keras.callbacks.TensorBoard(log_dir=log_dir, histogram_freq=1)
callbacks = [tensorboard_callback]

# Learn
history = model.fit(images_train, targets_train, epochs=10, validation_split=0.2, callbacks=callbacks)

Ce callback permet d'enregistrer des informations pendant l'apprentissage dans le fichier log_dir. Nous pouvons alors visualiser ces informations enregistrées en lançant TensorBoard.

tensorboard --logdir logs

Puis il suffit d'ouvrir la page http://localhost:6006/ dans un navigateur pour avoir accès à TensorBoard.

La première page Scalars représente l'évolution de l'accuracy et de l'erreur au cours de l'apprentissage.


La deuxième page Graphs représente le réseau de neurones utilisé.


La troisième page Distributions représente la distribution des biais et des poids appris.


La quatrième page Histograms représente la distribution des biais et des poids appris.


La dernière page Profile représente le temps passé dans chaque partie de l'apprentissage.
